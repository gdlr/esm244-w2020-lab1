Lets clean this data frame up so its tidy

How do we get rid of the dollar sign? tidyr::separate() then as.numeric() or readr::parse_number()

parse_number is awesome! use it within a mutate() parse_number only displays 7 sig figs, but doesnt actually get rid of any digits

landings_tidy <- us_landings %>% 
  mutate(state = str_to_lower(state),
         afs_name = str_to_lower(afs_name)) %>% 
  mutate(dollars_usd = (parse_number(dollars_usd,
                      na = c("NA"))))

Lets just get some salmon information

How do we remove data that we dont want, aka the aggregate species data? Use str_remove, and then specify by the string you want it to recognize. Careful, though, better have a good reason for ignoring that jk this doesnt remove data, just the word

We want data for just salmon. how do we filter by strings? use str_detect() If you use it in the beginning of a filter function, it only keeps things with that string.

How do we separate species from salmon? we can use separate()

salmon_landings <- landings_tidy %>% 
  mutate(afs_clean = str_remove(afs_name, " aggregate")) %>% 
  filter(str_detect(afs_clean, patter = "salmon")) %>% 
  separate(afs_clean, into = c("group", "species"), sep = ",")

Find some grouped summary data:

lets find the annual total US landings and dollar value, summing across all states, for each TYPE of salmon using group_by() and summarize()

salmon_summary <- salmon_landings %>% 
  group_by(year, species) %>% 
  summarize(
    tot_landings = sum(landings_pounds),
    tot_value = sum(dollars_usd))

Make a graph

salmon_landings_graph <- 
  ggplot(salmon_summary, aes(x = year, y = tot_landings)) +
  geom_line(aes(color = species)) +
